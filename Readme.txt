This is a small program from brute force on simple login page.
 If required download the missing python module.
____________________________________________________________________________

Brute force = consists of an attacker submitting many passwords or passphrases with the hope of eventually guessing a combination correctly. The attacker systematically checks all possible passwords and passphrases until the correct one is found.
_____________________________________________________________________________

Usage :
               -w : url
                -u : username
                 -t : threads
                 -f : password list
Example : python brutefpy.py -w http://abcxyz.com -u admin -t 4 -f pass.txt
